Instructions

  • Cliquez sur une boule pour la faire rouler
  • Utilisez "Lancer les boules!" pour une animation automatique
  • Le but est de se rapprocher du cochonnet (petite boule jaune)
  • Rouge vs Bleu - comme dans une vraie partie!
